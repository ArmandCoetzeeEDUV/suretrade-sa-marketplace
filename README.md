# SureTrade SA Marketplace

## Project Description

SureTrade SA is a Customer-to-Customer (C2C) e-commerce marketplace. The platform allows users to register, create product listings, browse products, search and filter listings, communicate with sellers, purchase items, and view their purchase history.

Administrators can manage users and product listings through a separate admin interface.

## Technologies Used

* PHP
* MySQL
* HTML
* CSS
* JavaScript
* XAMPP

## Installation Instructions

1. Install XAMPP.
2. Copy the project folder into the `htdocs` directory.
3. Start Apache and MySQL through the XAMPP Control Panel.
4. Open phpMyAdmin.
5. Create a database named `suretrade`.
6. Import the `database.sql` file included in this repository.
7. Update the database connection settings in `config.php` if required.
8. Open the project in a browser using:

   `http://localhost/suretrade-sa/`

## User Roles

* Consumer: Can register, create listings, browse products, message sellers, and purchase items.
* Administrator: Can manage users and product listings.
